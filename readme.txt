Task 11. varA:

Develop a multi-threaded application.
Use the capabilities provided by the java.util.concurrent package.
Do not use the word synchronized.
All entities that want to access the resource must be threads.
Use the capabilities of OOP.
Do not use the graphical interface. The application must be console.

1. The port. Ships enter the port to unload / load containers. The number of containers currently in the port and on the ship must be non-negative and exceed the specified ship's carrying capacity and port capacity. There are several berths in the port. One berth can have one ship. The ship can be loaded at the berth, unloaded, or both.
